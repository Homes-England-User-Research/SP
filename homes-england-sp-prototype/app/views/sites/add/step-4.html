{# -- Section: Start on site -- #}

Start on site

The start on site milestone applies as a payment event only to eligible schemes where start on site is achieved. It is triggered when:

  • the provider and the building contractor have entered into a building contract
  • the building contractor takes possession of the site or property
  • the works have commenced

Read more about the start on site milestone in the Capital Funding Guide, from 4.1 (opens in a new tab).

{# -- Section: Practical completion -- #}

Practical completion

Practical completion is when the last dwelling has been handed over, with all homes approved as fit for occupation in accordance with the relevant certifications and requirements. Read more about the practical completion milestone in the Capital Funding Guide, from 4.1 (opens in a new tab).
