Components

The following components extend the GOV.UK Design System for use in Homes England Strategic Partnerships. Each component follows WCAG 2.1 AA as a minimum, is keyboard navigable, and has been tested with screen readers. Where a component deviates from or extends a standard GDS pattern, the rationale is documented.
